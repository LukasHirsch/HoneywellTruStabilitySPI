Detailed Description

Class for reading temperature and pressure from a Honeywell TruStability HSC or SSC sensor.

Constructs a new presssure sensor object. min_pressure an max_pressure are taken from the datasheet and represent the 10% and 90% calibrated output pressures. Subsequent calls to pressure() will return values in the units of min_pressure and max_pressure.

Parameters
pinthe slave select pin of the sensor
min_pressurethe minimum calibrated output pressure
max_pressurethe maximum calibrated output pressure
spi_settingsSPI configuration settings. Default SPI settings use 800 KHz SPI

uint8_t TruStabilityPressureSensor::readSensor ( )
inline

Polls the sensor for new data. The raw temperature and pressure variables are updated. There is no guarantee that the data retrieved from the sensor is fresh data. Check the status.

Returns
The status of the sensor 0 indicates normal operation, 1 indicates the device is in command mode, 2 indicates stale data, 3 indicates a diagnostic condition
